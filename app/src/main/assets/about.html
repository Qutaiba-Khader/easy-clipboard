Easy Clipboard

A modern, no-root / no-Xposed clipboard manager.

Easy Clipboard is a clean-room reimplementation inspired by the now-discontinued Xposed app Native Clipboard by DHM47. It reuses the original's clip-card grid look while replacing the Xposed hooks with modern, supported Android entry points: the system PROCESS_TEXT action, an input-method clip tray, an accessibility paste helper, and a foreground clipboard monitor service. An optional Shizuku backend is scaffolded for privileged background capture.
